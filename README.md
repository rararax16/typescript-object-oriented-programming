# TypeScript × オブジェクト指向プログラミング

## 概要
TypeScriptの勉強した際に作成した健康管理アプリになります。  
TypeScriptはフロントjsのために使用しており、オブジェクト指向プログラミングで作成しています。  
手続き型と違いを知ることもできます。  

## set up

Dockerの起動
```
docker-compose up -d
```

Docker接続
```
docker-compose exec nodejs bash
```

TypeScriptのコンパイル
```
tsc
```

## wake up app
index.htmlをブラウザで表示してください。
